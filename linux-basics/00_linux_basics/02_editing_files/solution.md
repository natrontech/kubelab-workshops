# Solution for Editing Files with nano/vim

- Using nano:
  1. `cd ~/file-editing`
  2. `nano editme.txt`
  3. Make the necessary edits.
  4. `CTRL + O` then `Enter` to save.
  5. `CTRL + X` to exit.

- Using vim:
  1. `cd ~/file-editing`
  2. `vim editme.txt`
  3. Press `i` to enter insert mode.
  4. Make the necessary edits.
  5. Press `ESC`.
  6. Type `:wq` and press `Enter` to save and exit.
